## Установка

1.  **Установите зависимости:**

    ```bash
    npx create-react-app formsreact
    ```
2.  Установите `router-dom` для навигации:

    ```bash
    npm install react-router-dom
    ```

3.  **Запустите приложение:**

    ```bash
    npm start
    ```
